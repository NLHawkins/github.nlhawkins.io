Shoplist is built using boilerplate ASP.Net user authentication extended to include prefered shopping location and username authentication.





Once a user is logged in they have the ability to create and browse listings, filtering by location, parent category, or subcategory.





A user has mulitiple view and order optins for listings, and these can be quickly filtered through the top menu.

Description

ShopList was built as a weekendd project to practice duplicating various aspects of the popular Craigslist Web Application.
